# HessianFree_RNN
HessianFreeをRNNに適応することを目的にしています

# 実行方法

  `$python3 setup.py`
  
  `$docker start -i my-esn`
  
  通常のRNNの実行

  `$python3 src/train.py`
  
  hessianfree法のRNN適応の実行
  
  `$python3 src/hessian_train.py`
